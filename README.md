# Jarbas
"O Jarbas é uma ferramenta de sistema que permite a gestão e ativação de permissões elevadas para outros aplicativos sem a necessidade de Root. Utilizando a interface Shizuku/ADB, ele fornece acesso seguro a APIs restritas do Android, garantindo que as automações e permissões permaneçam ativas de forma estável e ininterrupta."
o Jarbas utiliza o ADB Wireless para funcionar sem root.
